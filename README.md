# HR-Sales-Data-Dashboard(Power BI)

## Project Overview
This Power BI dashboard provides detailed insights into employee data for HR management.  
It helps visualize headcount, salary distribution, gender diversity, performance ratings, and departmental trends** to support data-driven decisions.

## Data used
Dataset - (https://github.com/Neha-sam/HR-Sales-Data-Dashboard/blob/main/hr-data%20(1).xlsx)

## Questions (KPI's)
- Analyze employee performance ratings.
- Compare salary ranges across departments.
- Understand workforce distribution by age, gender, and department.
- Monitor hiring trends and total salary expenses.

## Process
Imported HR dataset from Excel into Power BI.
2. Cleaned data (removed nulls, formatted dates, converted salary to numeric).
3. Created DAX measures:
   - Total Salary = SUM(Salary)
   - Average Salary = AVERAGE(Salary)
   - Headcount = COUNT(EmployeeID)
4. Designed visuals:
   - Bar, Column, Line, and Scatter charts.
   - KPIs and Slicers for Country and Alphabet filters.
